# Custom 3D & 4D Icons Directory

Place your custom 3D/4D rendered icons (transparent `.png` format) in this folder. The game will automatically detect them and display them in place of the default icons and emojis!

## Expected Filenames & Sizes

Please name your transparent PNG files exactly as shown below:

### Navigation Tabs (Bottom Nav)
* `nav_friends.png` (Friends tab)
* `nav_daily_bonus.png` (Daily Bonus tab)
* `nav_home.png` (Home tab)
* `nav_daily_task.png` (Daily Task tab)
* `nav_store.png` (Store tab)

### Main Play Banners (Lobby)
* `play_online.png` (Play Online banner)
* `play_friends.png` (Play With Friends banner)
* `play_practice.png` (Practice banner)

### Action Buttons & Utilities
* `action_mail.png` (Mail envelope button)
* `action_chat.png` (Chat button in header)
* `action_settings.png` (Settings gear button in header)
* `coin.png` (Coin icon used in stats/balance capsule)
* `action_share.png` (Invite/Share button)
* `action_ads.png` (Watch Ads button)

### Tactics Cards (Boosters)
* `tactic_shield.png` (Chip Guard booster)
* `tactic_wildUpgrade.png` (Wild Upgrade booster)
* `tactic_reroll.png` (Card Redraw booster)
* `tactic_spy.png` (Spying Glass booster)
* `tactic_emp.png` (Shield Breaker booster)
* `tactic_handExchange.png` (Hand Exchange booster)

## Recommended Specifications
* **Format**: Transparent PNG or WebP
* **Dimensions**: 128x128 pixels (square ratio)
* **Design Style**: Sleek 3D renders with matching color schemes (gold, cyan, purple) and shiny/reflective materials to complement the pseudo-3D look.
